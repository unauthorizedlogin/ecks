# Code of Conduct

## Our Commitment

The Ecks RPG Engine community is dedicated to providing a welcoming, respectful, and collaborative environment for developers, creators, testers, and community members.

We value constructive discussion, creative collaboration, and shared learning.

---

# Expected Behavior

Community members are expected to:

- Be respectful and professional
- Provide constructive feedback
- Help others learn and improve
- Respect different experience levels
- Discuss ideas and technical decisions in good faith
- Credit the work of others
- Follow platform-specific rules and guidelines

---

# Unacceptable Behavior

The following behavior is not acceptable:

- Harassment or personal attacks
- Discrimination or hateful conduct
- Threats or intimidation
- Deliberate disruption of discussions
- Sharing private information without permission
- Malicious behavior toward the project or community
- Excessive hostility during disagreements

Technical disagreements are expected and encouraged. Personal attacks are not.

---

# Development Discussions

This project encourages open discussion about:

- Architecture decisions
- Feature ideas
- Implementation approaches
- Documentation improvements
- Bug reports

Different approaches and opinions are welcome.

Feedback should focus on improving the project rather than criticizing individuals.

---

# Discord Community

The official Discord community follows this Code of Conduct.

Additional Discord-specific rules may exist to help organize discussions and maintain a healthy community environment.

Members should:

- Use appropriate channels
- Avoid unnecessary spam
- Respect moderators and community members
- Keep discussions relevant to the channel topic

---

# Enforcement

Violations of this Code of Conduct may result in:

- A warning
- Removal of inappropriate content
- Temporary restrictions
- Removal from community spaces

Actions will be taken based on the severity and frequency of violations.

---

# Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, report it through:

- Discord moderation channels
- Project maintainers
- Appropriate GitHub reporting tools

Reports will be reviewed as fairly and privately as possible.

---

# Scope

This Code of Conduct applies to all official Ecks RPG Engine community spaces, including:

- GitHub repositories
- Discord community spaces
- Future official communication channels

---

Thank you for helping build a positive and collaborative community around the Ecks RPG Engine.
